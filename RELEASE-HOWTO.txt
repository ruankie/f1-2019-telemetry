=========== Release procedure:

(1) Edit docs/source/conf.py, README.md, and setup.py to bump version number.

    Note that in README.md, the version number occurs multiple times.

(2) Do a commit of the current state of the repository, and tag it.

(3) Run the 'make-dist.sh' script.

(4) Run the 'upload-dist.sh' script.
